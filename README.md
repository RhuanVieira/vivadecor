VivaDecor – Landing Page Responsiva em PHP

O VivaDecor é uma landing page desenvolvida utilizando PHP, HTML e CSS, com foco em responsividade e organização clara dos arquivos.
O projeto não utiliza banco de dados e não possui backend funcional; o PHP é usado apenas para estruturar o conteúdo da página.

1. Tecnologias Utilizadas

PHP (apenas para estrutura)

HTML5

CSS3

JavaScript

Git e GitHub

2. Como Rodar o Projeto

Por utilizar PHP, é necessário rodar em um servidor local.

Usando XAMPP

Coloque a pasta do projeto em:

C:\xampp\htdocs\VivaDecor


Inicie o Apache no painel do XAMPP.

Acesse no navegador:

http://localhost/VivaDecor

Usando o servidor embutido do PHP

Dentro da pasta do projeto, execute:

php -S localhost:8000


E acesse:

http://localhost:8000
